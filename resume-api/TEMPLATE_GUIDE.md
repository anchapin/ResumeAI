### Layout Options

#### Margin Settings (inches)

- Left: 0.25 - 2.0 (default: 0.75)
- Right: 0.25 - 2.0 (default: 0.75)
- Top: 0.25 - 2.0 (default: 0.6)
- Bottom: 0.25 - 2.0 (default: 0.6)

**Recommendations**:

- Tighter margins (0.5"): More content, less white space
- Standard margins (0.75"): Balanced, professional
- Wider margins (1.0+"): More breathing room, executive feel

#### Paper Size Options

- **Letter**: Standard US (8.5" x 11") - default
- **A4**: International standard (210mm x 297mm)

**Note**: A4 may require slight content adjustments for optimal layout.

## Color Schemes

### How to Use Color Schemes

Color schemes define the visual identity of your resume:

- **Primary**: Main text and headers (usually dark)
- **Accent**: Emphasis elements, links, highlights
- **Secondary**: Subtext, dates, supporting information

### Best Practices

1. **Match Industry Standards**
   - Technical: Cool blues, greens
   - Creative: Bold, distinctive colors
   - Executive: Traditional navy, burgundy
   - Academic: Classic blacks, blues

2. **Maintain Contrast**
   - Ensure text is readable (dark primary colors)
   - Use accent colors sparingly for emphasis
   - Test color combinations for accessibility

3. **Corporate Branding**
   - Match company colors when applying
   - Use subtle variations for professionalism
   - Consider company culture (conservative vs. modern)

4. **Print-Friendly**
   - Dark colors print well
   - Avoid very light colors for text
   - Test print output if possible

## ATS Optimization

### What is ATS?

Applicant Tracking Systems (ATS) are software used by companies to:

- Scan and parse resumes
- Extract key information
- Match candidates to job requirements
- Filter and rank applications

### ATS-Friendly Features

All ResumeAI templates are designed with ATS optimization:

✅ **Standard Fonts**: Helvetica, Times New Roman, Arial
✅ **Simple Layouts**: No complex tables or graphics
✅ **Clear Section Headers**: Standard section names
✅ **Text-Based**: No images or special formatting
✅ **Standard Spacing**: Normal margins and line spacing
✅ **Keyword-Friendly**: Easy scanning by ATS algorithms

### ATS Best Practices

1. **Use Standard Section Names**
   - "Experience" instead of "My Career Journey"
   - "Education" instead of "Learning"
   - "Skills" instead of "My Superpowers"

2. **Include Keywords**
   - Match job description keywords
   - Use industry-standard terminology
   - Include both acronyms and full terms

3. **Keep It Simple**
   - Avoid excessive formatting
   - Don't use columns or tables
   - Skip graphics and images

4. **Test ATS Parsing**
   - Save as PDF or plain text
   - Upload to ATS test tools
   - Verify content extraction

## Best Practices

### Content Organization

1. **Section Order**
   - Contact info (always first)
   - Professional summary
   - Experience (reverse chronological)
   - Education
   - Skills
   - Additional sections (projects, awards, etc.)

2. **Length Guidelines**
   - Entry-level: 1 page
   - Mid-level: 1-2 pages
   - Senior/Executive: 2 pages
   - Academic: Variable (often 2-4 pages)

3. **Bullet Points**
   - Start with action verbs
   - Quantify achievements
   - Focus on results, not responsibilities
   - Keep concise (1-2 lines each)

### Visual Design

1. **White Space**
   - Don't overcrowd the page
   - Use consistent spacing
   - Leave margins for notes/printing
   - Ensure readability

2. **Hierarchy**
   - Name/Title largest
   - Section headers medium
   - Body text standard size
   - Dates/subtitles smaller

3. **Consistency**
   - Same formatting for similar elements
   - Consistent date formats
   - Uniform bullet point style
   - Aligned sections

### Industry-Specific Tips

#### Tech Industry

- Highlight technical skills prominently
- Include relevant projects
- Mention specific technologies
- Quantify impact (e.g., "improved performance by 40%")

#### Creative Industry

- Showcase portfolio links
- Emphasize creative achievements
- Include relevant tools/software
- Balance creativity with professionalism

#### Executive Roles

- Focus on leadership and strategy
- Highlight team accomplishments
- Include company growth metrics
- Demonstrate vision and impact

#### Academic Positions

- List publications prominently
- Include research grants
- Highlight teaching experience
- Mention academic service

#### Healthcare

- Emphasize certifications
- Include clinical experience
- Highlight patient outcomes
- List specialized training

## API Usage

### List All Templates

```bash
curl -X GET "https://api.resumeai.com/v1/templates" \
  -H "X-API-KEY: your_api_key"
```

### Filter by Category

```bash
curl -X GET "https://api.resumeai.com/v1/templates?category=technical" \
  -H "X-API-KEY: your_api_key"
```

### Get Template Metadata

```bash
curl -X GET "https://api.resumeai.com/v1/templates/technical" \
  -H "X-API-KEY: your_api_key"
```

### Generate PDF with Template

```bash
curl -X POST "https://api.resumeai.com/v1/render/pdf" \
  -H "X-API-KEY: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "resume_data": {
      "basics": {
        "name": "John Doe",
        "email": "john@example.com"
      }
    },
    "variant": "technical"
  }'
```

### Preview Template

```bash
curl -X POST "https://api.resumeai.com/v1/templates/technical/preview" \
  -H "X-API-KEY: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "resume_data": {
      "basics": {
        "name": "John Doe",
        "email": "john@example.com"
      }
    }
  }'
```

### Customize Template

```bash
curl -X POST "https://api.resumeai.com/v1/render/pdf" \
  -H "X-API-KEY: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "resume_data": {
      "basics": {
        "name": "John Doe",
        "email": "john@example.com"
      }
    },
    "variant": "technical",
    "customization": {
      "color_scheme": "blue",
      "font": "Arial",
      "paper_size": "letter",
      "margin_left": 0.75,
      "margin_right": 0.75,
      "margin_top": 0.6,
      "margin_bottom": 0.6
    }
  }'
```

## Troubleshooting

### Common Issues

**Issue: PDF looks different than expected**

- Solution: Ensure you're using the latest template version
- Check that customization options are valid
- Verify color scheme name matches template metadata

**Issue: Content doesn't fit on page**

- Solution: Adjust margins to be smaller (0.5" minimum)
- Reduce bullet point length
- Remove less relevant sections
- Consider using the Technical template for more space

**Issue: Formatting looks off**

- Solution: Check that resume data follows JSON Resume format
- Ensure proper character encoding (UTF-8)
- Validate that no special characters cause issues

**Issue: ATS parsing problems**

- Solution: Use Base or Technical template
- Ensure standard fonts (Helvetica, Times New Roman)
- Check for proper section headers
- Test with ATS parsing tools
